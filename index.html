YouTube Analytics ELT Pipeline with Airflow

Built an ELT pipeline to extract 9K+ video records via YouTube API, store raw data in GCS, and load it into BigQuery. Provisioned GCP infrastructure using Terraform

Data Engineering

Integrated Databricks ETL Pipeline and Azure CI/CD Deployment (Part 1)

Developed an ETL pipeline to efficiently process batch data using Databricks Auto Loader within the Medallion Architecture. Terraform automates full infrastructure provisioning (Storage, Databricks, Key Vault), while Airflow simulates batch data processing. Power BI is used to visualize and analyze data from the Gold layer stored in Delta tables within Databricks using the JDBC/ODBC connector

Technologies: Airflow, Azure, Terraform, Databricks, Power BI

End-to-End Azure ETL Pipeline with Terraform

Built an ETL pipeline to extract 56K+ records from SQL Server using Azure Data Factory, store in ADLS Gen2, and transform via Databricks (Medallion Architecture).
